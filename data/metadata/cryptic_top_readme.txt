CRyPTIC README, 14th October 2021


This ftp site contains data released by the CRyPTIC project in September 2021.
The directory structure is as follows:

There are two top-level directories:

reuse/
    Look here if you want to get CRyPTIC project data (genetic and/or
    phenotypic) and use them for your own purposes. 
    
    If you do publish using this data, please do cite our Data paper, currently
    a preprint:

    The CRyPTIC Consortium, “A data compendium of Mycobacterium tuberculosis
    antibiotic resistance”, https://doi.org/10.1101/2021.09.14.460274

reproducibility/
    This contains the data used for multiple CRyPTIC project publications. 

    Each paper has taken slightly different subsets of the data and filtered
    slightly differently, as documented in those papers.
